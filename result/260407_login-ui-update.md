# [로그인/회원가입 진입 페이지 개편]

## What I Did
- `Home.tsx`의 기존 '교사 로그인' 메인 버튼을 '로그인/회원가입'으로 텍스트 및 역할 수정
- 여러 사용자가 통합적으로 로그인/회원가입 옵션을 선택할 수 있는 `Entry.tsx` 신규 생성
  * 상단에 [학생], [교사], [일반회원] 세 가지 로그인 권한 아이콘 버튼 분리 배치
  * 전체 너비(Full-width)의 하단 가로형 [회원가입] 버튼 추가
- `App.tsx` 라우팅 업데이트 (`/teacher-login` -> `/entry`)

## Issues
- 추가된 회원가입 버튼의 배경색을 초기에 회색 계열(`bg-slate-100`)로 설정했을 때, 전체 페이지 배경(`bg-slate-50`)과 대비가 적어 구분이 안 되고 묻히는 디자인 이슈 발생

## Solutions
- 기존 버튼들이 가지는 푸른색 계열의 파스텔 톤과 자연스럽게 어울리면서 동시에 가시성을 높일 수 있는 **Cyan 파스텔 톤**(`bg-cyan-100`, `shadow-cyan-200/50`)으로 배경색과 그림자 수정
- 내부 아이콘에도 통일감을 위해 `text-cyan-500` 클래스 적용

## Next Steps
- 각 사용자 유형별(학생, 교사, 일반) 로그인 기능 상세 연동
- 통합 회원가입 폼(`/signup`) 페이지 UI 구현 및 파라미터 연결 기능 등 개발 진행
